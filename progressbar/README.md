# Progressbar example

Simple example of how to use progressbar from schollz.

```sh
go get github.com/schollz/progressbar
```

Now run the program.

```sh
go run progressbar.go
```

```txt
Run first loop: not complete
  60% |########################================| [2s:1s]            Break.
Run second loop: complete
 100% |########################################| [4s:0s]            Done.
```
